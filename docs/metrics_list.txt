# CourseScope - Metrics file-only (v1.1.48)

Toutes les metriques ci-dessous sont calculees a partir d'un seul fichier GPX/FIT uploade.
Certaines metriques sont conditionnelles aux champs disponibles (ex: HR, power, cadence).

Legend:
- [Both] GPX ou FIT
- [FIT] necessite un FIT avec champs presents
- [Cond] champ optionnel dans le fichier
